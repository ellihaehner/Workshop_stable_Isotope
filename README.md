# Workshop_stable_Isotope
This repository contains files to create C-N-scatterplot, as well as C-O-plots for an incremental analysis as shown at the Stable Isotope Workshop in March 2026. 
## Contents
There are two folders.

## C-N-Collagen-graphs
The C-N-Collagen-graphs folder contains a script to plot C-N-values as a scatterplot. While the first two diagrams plot 
species with different points and colours. The third plot, will plot animal silhouettes. 
For further details see readme in Ordner C-N-Collagen-graphs. 
The Code was created by E. Hähner together with J. Göbel.

## Iso_code
The Iso_code folder contains a script for plotting the incremental data. The plot will create Line graphs with the Oxygen and 
Carbon values of animal teeth samples.
